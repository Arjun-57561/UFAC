## 🚨 TROUBLESHOOTING

### If PDF won't insert:
- Try PNG version instead
- Check Word version (2016+ recommended)
- Try "Insert Object" instead of "Insert Picture"

### If figure is blurry:
- Make sure you're using PDF, not PNG
- Don't resize too much (keep 75-100%)
- Check "High Quality" in Word's image options

### If text doesn't flow:
- Check for extra line breaks
- Verify paragraph spacing
- Use "Show/Hide" (¶) to see formatting marks

### If cross-references break:
- Use Word's "Insert Cross-reference" feature
- Update all fields (Ctrl+A, then F9)
- Check figure numbering is sequential
